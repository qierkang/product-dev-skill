# Requirement Gate

## 必备文件

- `00-需求总览.md`
- `需求文档.md`
- `manifest.json`

## 最低检查点

### 需求总览

- 必须包含：`输入资产`
- 必须包含：`页面清单`
- 必须包含：`待确认项`

### 需求文档

- 必须包含：`业务背景`
- 必须包含：`核心目标`
- 必须包含：`功能模块总览` 或 `功能模块`
- 必须包含：`表单定义`
- 必须包含：`业务规则`

### manifest

- 必须包含：`request_key`
- 必须包含：`title`
- 必须包含：`owner`
- 必须包含：`current_stage`

## 通过标准

- 三个文件齐全
- 文档不是纯模板占位
- 至少能支撑产品内部评审
